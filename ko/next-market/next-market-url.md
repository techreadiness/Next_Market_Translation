# Next Market 주요 URL 안내

서비스 접속 URL부터 비즈니스 대시보드, 관리자 콘솔까지 **NEXT Market 이용 및 개발·운영에 필요한 주요 접근 경로를 안내드립니다.**

## NEXT Market

* 게임 개발사와 Web2 게이머에게 새로운 자유 경제의 경험을 제공하는 글로벌 크로스보더 게임 마켓입니다.
* NEXT Market에 온보딩된 게임 마켓을 통해 다양한 아이템 거래 및 서비스를 경험할 수 있습니다.

{% embed url="https://www.nextmarket.games/" %}

## Biz Dashboard

* Biz Dashboard는 매출, 정산 등 주요 비즈니스 지표에 대한 통계 및 분석 데이터를 제공합니다.
* Biz Dashboard 접근을 위해서는 **사전 권한 부여가 필요**합니다.
* 온보딩 진행 시, **회사 메일 주소를 사전에 NEXT Market 측에 전달**해 주셔야 하며 해당 메일 주소로 계정 생성 및 가입이 가능합니다.

{% embed url="https://biz.nextmarket.games/" %}

## NEXT Market Console

* NEXT Market Console은 **NEXT Market에서 제공하는 주요 API 기능을 웹 기반 콘솔 형태로 제공하는 관리자 도구**입니다.
* 개발 및 운영 과정에서 **아이템, 상품, 미션 등을 쉽고 직관적으로 등록·관리**할 수 있도록 지원합니다.
* Console 접근을 위해서는 **콘솔 전용 API Key**가 필요합니다.
* Console 사용을 희망하시는 경우, **사전에 NEXT Market 측으로 요청**해 주시면 확인 후 **전용 API Key 발급을 지원**해 드립니다.

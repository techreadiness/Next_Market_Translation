# Key URLs for Next Market

From service access URLs to the Business Dashboard and Admin Console, **we provide key access paths necessary for using, developing, and operating NEXT Market.**

## NEXT Market

* A global cross-border game marketplace offering a new free economy experience for game developers and Web2 gamers.
* Experience diverse item trading and services through games onboarded to NEXT Market.

{% embed url=&quot;https://www.nextmarket.games/&quot; %}

## Biz Dashboard

* Biz Dashboard provides statistical and analytical data on key business metrics like sales and settlements.
* **Prior authorization is required** to access Biz Dashboard.
* During onboarding, you must **provide your company email address to NEXT Market in advance**. Account creation and registration are possible using this email address.

{% embed url=&quot;https://biz.nextmarket.games/&quot; %}

## NEXT Market Console

* NEXT Market Console is an **administrative tool that provides key NEXT Market API functions via a web-based console**.
* It supports **easy and intuitive registration and management of items, products, missions, etc.** during development and operations.
* Access to the Console requires a **dedicated Console API Key**.
* If you wish to use the Console, **please request it from NEXT Market in advance**. After verification, we will **assist with issuing a dedicated API Key**.

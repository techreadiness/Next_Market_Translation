# 아이템 바로 구매하기

## 아이템 바로 구매하기 기능

아이템 바로 구매하기 기능은 게임 플레이 중 필수 소모성 아이템이 부족하거나 게임 내에서 아이템 구매를 선택하는 경우, NEXT Market의 결제 화면을 즉시 제공하여 아이템을 구매하고 게임 플레이를 중단 없이 이어갈 수 있도록 지원하는 기능입니다.

## 적용 대상 아이템 및 환경별 제공 방식

### 대상 아이템

해당 기능은 게임 진행에 필수적인 소모성 아이템에 적합합니다.&#x20;

* 적용 아이템 예시
  * 부활권, 경험치 복구권, 스테미나 회복 아이템, 재도전 티켓 등 게임 진행을 이어가기 위한 기타 소모성 아이템
* 소모성 아이템 외에도, 게임에서 판매하는 모든 아이템에 적용이 가능합니다.

### 환경별 제공 방식

아이템 바로 구매하기 기능은 게임 실행 환경에 따라 결제 화면 제공 방식이 상이합니다.

## 사용자 동선

### 1. 아이템 바로 구매하기 적용 동선

아이템 바로 구매하기 기능 적용 시, 아래와 같은 간결한 동선으로 결제가 진행됩니다.

* 동선
  * 게임 플레이 > 구매 아이템 및 구매수량 선택 > NEXT Market 결제 화면으로 이동 > 결제 진행 > 결제 완료 후 게임 복귀
* 효과
  * 기존 동선 대비 유저 이동 간계 최소화
  * API 설정 등의 운영 관점에서 추가 설정 공수 최소화 가능

### 2. 기존 동선 (아이템 바로 구매하기 미적용 시)

아이템 바로 구매하기 기능을 적용하지 않는 경우, NEXT Market을 아래와 같은 동선으로 이용할 수 있습니다.

<table><thead><tr><th width="193.2725830078125">Case</th><th>Flow</th></tr></thead><tbody><tr><td>NEXT Market 이동</td><td><ul><li>게임 플레이 > NEXT Market 이동하기 > 아이템 리스트 탐색 > 아이템 선텍 > 아이템 상세 화면 > 구매 아이템 및 구매 수량 선택 > 결제 진행 > 결제 완료 후 NEXT Market 복귀</li><li>게임 플레이 > NEXT Market 이동하기 > 아이템 리스트 탐색 > 구매 아이템 및 구매 수량 선택 > 결제 진행 > 결제 완료 후 NEXT Market 복귀</li></ul></td></tr><tr><td>아이템 상세 화면 이동</td><td>게임 플레이 > 구매 아이템 선택 > NEXT Market 아이템 상세 화면 > 구매 아이템 및 구매 수량 선택  > 결제 진행 > 결제 완료 후 NEXT Market 복귀</td></tr></tbody></table>

⚠️ 아이템 상세 화면 이동 동선 제공 시 유의사항&#x20;

스토어 인증용 원타임 토큰 요청 API 호출 시 redirectPath에 아래 형식을 설정해야 아이템 상세 화면 이동 기능이 활성화됩니다.

```
/shop/sales/{sale_id}
```

### 3. 아이템 바로 구매하기 vs 기존 동선 비교

| 구분        | 아이템 바로 구매하기 | 기존 동선 (NEXT Market 이동) | 기존 동선 (아이템 상세 화면 이동) |
| --------- | ----------- | ---------------------- | -------------------- |
| 아이템 탐색 경로 | 인게임         | NEXT Market            | 인게임                  |
| 유저 이동 단계  | 최소화         | 다단계                    | 다단계                  |
| 결제 접근     | 즉시          | 아이템 상점 리스트 경유          | 아이템 상세 화면 경유         |
| 게임 이탈 가능성 | 낮음          | 상대적으로 높음               | 상대적으로 높음             |
| 운영 설정 공수  | 낮음          | 낮음                     | 상품 등록 시 마다 설정 필요     |

## 아이템 바로 구매하기 적용 방법

아이템 바로 구매하기 기능 적용을 희망하는 경우 LINE NEXT와 사전 협의 후 적용이 가능합니다.\
적용 대상 아이템, 게임 환경 및 동선 구조에 따라 세부 적용 방식은 협의를 통해 결정됩니다.
